# Smart Genealogy

A cross-platform .NET MAUI application for managing genealogical data. This app allows users to view, manage, and interact with genealogical information across Windows and macOS platforms.

## Project Status: Work in Progress

This application is still in development. Core functionality is being implemented, and features may change.

## Tech Stack

- .NET 9 / .NET MAUI
- C#
- MVVM Architecture
- Community Toolkit MAUI
- Community Toolkit Markup
- Target Platforms: Windows, macOS
